# **Инструкция**
1. Скопировать репозиторий
2. Открыть папку с файлами в VSC
3. Установить зависимость: `pip install requests python-dotenv`
4. Создать файл `.env` и добавить строчку `API_KEY = (ваш API-ключ)`
5. запустить файл `main.py`
